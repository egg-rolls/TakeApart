# TakeApart 产品需求文档 (PRD)

> 版本: 2.0 | 日期: 2026-07-03 | 状态: Draft

---

## 1. 产品概述

### 1.1 产品名称
TakeApart — AI 视觉拆解工具

### 1.2 一句话定义
让 AI 拥有 PS 级别的"眼睛和手"，通过 MCP 协议对图片进行像素级元素拆解。

### 1.3 产品形态
MCP Tool Server，stdio 本地部署，供视觉识别 AI 调用。

---

## 2. 目标用户

| 用户角色 | 使用方式 | 核心诉求 |
|---------|---------|---------|
| **AI 模型** (Claude/GPT) | MCP 协议调用 | 精确拆解图片元素 |
| **无视觉能力的 AI** | detect fallback | 快速发现图中大致元素 |
| **AI 应用开发者** | 集成到 AI 工作流 | 为 AI 补充视觉操作能力 |

---

## 3. 核心能力 (2 个工具)

### 3.1 detect_elements — 轻量元素检测

**定位**：无视觉能力 AI 的 fallback，快速发现图中大致元素区域

| 属性 | 说明 |
|------|------|
| 内核 | Canny 边缘检测 + 轮廓分析（纯传统 CV） |
| GPU | 不需要 |
| 输入 | 图片路径 + 可选可视化输出路径 |
| 输出 | 元素列表 (bbox, area, contour) + 可选标注图 |
| 速度 | 极快 (< 1s) |
| 精度 | 粗粒度，适合分散元素 |

### 3.2 segment_element — 精确分割

**定位**：PS 式框选，硬边缘精确分割

| 属性 | 说明 |
|------|------|
| 内核 | SAM 粗分 + Canny 边缘精修 |
| GPU | 需要 (SAM 2) |
| 输入 | 框选坐标列表 (支持批量) |
| 输出 | mask base64 / 透明 PNG / 两者（参数切换） |
| 速度 | 单框 ~2s (CPU ~10s) |
| 精度 | 像素级，硬边缘 |

---

## 4. 成功指标

| 指标 | 目标值 |
|------|--------|
| 边缘质量 | 硬边缘，无模糊过渡 |
| 分割完整性 | 框内最大完整元素，非碎片 |
| 单框处理时间 | GPU < 3s, CPU < 15s |
| MCP 连接成功率 | 100% |
